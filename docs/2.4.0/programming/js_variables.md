## JS Variables
Within heropacks, we utilize vars, a keyword that automatically detects the data type you provide to it

### General Basics

```js
// The general basics to know before starting are the following:
var integer = 4; // Whole numbers
var double = 4.122345534543354; // higher precision
var float = 4.749353432; // less precision, but more better memory accuracy 
var bool = true; // Accepts false or true, is seen as 0 or 1 in binary
var char = 'a'; // A single character
var string = "apple"; // A collection of chars
```

### Advanced Data Types
